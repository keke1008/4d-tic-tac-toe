# 4D Tic-Tac-Toe - Tesseract Game

4次元空間でプレイする神秘的な三目並べゲーム。回転するテッセラクト（超立方体）の中でゲームを楽しめます。

## 🌌 特徴

- **完全な4次元ゲーム**: 4x4x4x4 = 256個のセルからなる真の4次元三目並べ
- **テッセラクト可視化**: リアルタイムで回転する4次元超立方体の美しい投影
- **4D回転**: XY, XZ, XW, YZ, YW, ZW の6つの回転平面
- **神秘的なビジュアル**: サイバーパンク風のネオンエフェクトとアニメーション
- **完全な勝利判定**: 4次元空間での全ての可能な方向（80方向以上）での4目並びを検出

## 🎮 遊び方

### 基本ルール

1. プレイヤーXとOが交互に4次元グリッドにマークを配置
2. 4次元空間で4つ並べたプレイヤーが勝利
3. 4x4x4x4 = 256個全てのセルが埋まると引き分け

### 操作方法

- **クリック**: 現在のW層のセルにマークを配置
- **W軸 ← / →**: 4次元の異なる層（W座標）を移動
- **回転ボタン**: 各平面でテッセラクトを回転
- **自動回転**: テッセラクトの自動回転をON/OFF
- **リセット**: ゲームを最初から

### 4次元の理解

ゲームボードは4つの次元で構成されています：

- **X軸**: 左右
- **Y軸**: 上下
- **Z軸**: 奥行き（重なって表示される層）
- **W軸**: 第4次元（W軸ボタンで切り替え）

画面には現在のW座標における3次元の層（4x4x4のグリッド）が表示されます。
Z軸は視覚的に重なった層として表現されています。

### 勝利条件

4次元空間では以下のような並び方で勝利できます：

- **3次元での勝利**: 同じW層内でX, Y, Z軸のいずれかの方向に4つ並べる
- **4次元対角線**: X, Y, Z, W全ての軸が同時に変化する完全な4次元対角線
- **その他80以上の方向**: 4次元空間では3^4 - 1 = 80通りの方向が存在

## 🔮 テッセラクトについて

テッセラクト（tesseract）は4次元の超立方体です：

- **頂点**: 16個（2^4）
- **辺**: 32本
- **面**: 24個の正方形
- **立方体**: 8個の3次元立方体で構成

画面上で見える回転するワイヤーフレームは、4次元のテッセラクトを3次元に投影し、
さらに2次元のスクリーンに投影した姿です。

## 🎨 技術詳細

- **投影**: 4D → 3D → 2D の二段階透視投影
- **回転**: 6つの回転平面での4次元回転行列
- **レンダリング**: HTML5 Canvas での60fps アニメーション
- **エフェクト**: 深度ベースの色相変化、グローエフェクト

## 📐 4次元回転の数学

4次元空間では6つの独立した回転平面が存在します：

- **XY平面**: 通常の2D回転
- **XZ平面**: 前後の回転
- **YZ平面**: 横方向の回転
- **XW平面**: X軸と第4次元の回転
- **YW平面**: Y軸と第4次元の回転
- **ZW平面**: Z軸と第4次元の回転

これらを組み合わせることで、3次元では不可能な複雑な回転が実現できます。
